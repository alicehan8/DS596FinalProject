## 🔧 Installation

Clone the repository:

```bash
# Clone
git clone https://github.com/alicehan8/DS596FinalProject.git
cd DS596FinalProject

# Create environment
conda create -n EpiModX python=3.10 -y
conda activate EpiModX
pip install -r requirements.txt
pip install mamba_ssm          # requires CUDA GPU

# Install parallel_experts (MoE routing — not on PyPI)
git clone https://github.com/UMass-Foundation-Model/Mod-Squad.git
cd Mod-Squad/parallel_linear
pip install --no-build-isolation .
cd ../..

```

## Downloading the Datasets

The human reference genome (e.g., GRCh38) can be downloaded from GENCODE and ENCODE (https://www.encodeproject.org/data-standards/reference-sequences) or run the following commands
```bash
wget https://hgdownload.soe.ucsc.edu/goldenPath/hg38/bigZips/hg38.fa.gz
gunzip hg38.fa.gz
samtools faidx hg38.fa
export REFERENCE_GENOME_PATH=/path/to/hg38.fa
```
AD datasets can all be found under the Datasets folder on the Zenodo project (https://zenodo.org/records/17528247)  
AML datasets can be generated by running the ```generate_dataset_AML.py``` file 

## ✅ Training LLM + MoE 

To start training the model, run:

```bash

```

```bash

```
